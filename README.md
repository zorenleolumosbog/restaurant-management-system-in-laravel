# Laravel Restaurant Management System

The Restaurant Management System Project is a PHP, Laravel, and MySQL database based web application. A Laravel application for a simple restaurant management system. The user can examine restaurant menus, dishes, and any other type of offer here. The user can also send an invitation to book a table at a restaurant.
